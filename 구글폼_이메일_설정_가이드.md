# 구글 폼 이메일 알림 설정 가이드

구글 폼 제출 시 `contact@jejusanbang.com`으로 자동 이메일 알림을 받는 방법입니다.

## 방법 1: 구글 폼 기본 알림 설정 (가장 간단)

1. 구글 폼을 엽니다: https://docs.google.com/forms/d/e/1FAIpQLSfK4leyAkl_X1dJWKrk5u35qa5QFXw_vF91iQuH6sp1PKqeqw/edit

2. 우측 상단의 **"응답"** 탭을 클릭합니다.

3. **"응답 알림 받기"** 토글 버튼을 켭니다.

4. **"이메일"** 옵션을 선택하고 `contact@jejusanbang.com`을 입력합니다.

5. 저장하면 새로운 응답이 제출될 때마다 이메일 알림을 받습니다.

## 방법 2: Google Apps Script를 사용한 자동 이메일 전송 (더 상세한 정보)

방법 1보다 더 상세한 이메일 본문을 받고 싶다면 Google Apps Script를 사용할 수 있습니다.

### 설정 단계:

1. **구글 폼 열기**
   - https://docs.google.com/forms/d/e/1FAIpQLSfK4leyAkl_X1dJWKrk5u35qa5QFXw_vF91iQuH6sp1PKqeqw/edit

2. **Apps Script 열기**
   - 우측 상단의 "⋯" (점 3개) 메뉴 클릭
   - **"스크립트 편집기"** 선택
   - 또는 직접 https://script.google.com 접속

3. **코드 입력**
   - `google-form-email-script.js` 파일의 내용을 복사하여 Apps Script 편집기에 붙여넣습니다.

4. **트리거 설정**
   - 좌측 메뉴에서 **"트리거"** 클릭
   - 우측 하단의 **"+ 트리거 추가"** 클릭
   - 설정:
     - 이벤트 소스: **"양식에서 제출"**
     - 이벤트 유형: **"양식 제출 시"**
     - 함수: **"onFormSubmit"**
   - 저장 클릭

5. **권한 승인**
   - 처음 실행 시 권한 승인 요청이 나타납니다.
   - **"권한 검토"** 클릭
   - Google 계정 선택
   - **"고급"** → **"안전하지 않은 페이지로 이동"** 클릭
   - **"허용"** 클릭

6. **테스트**
   - 구글 폼에 테스트 응답을 제출해봅니다.
   - `contact@jejusanbang.com`으로 이메일이 도착하는지 확인합니다.

### 이메일 형식

이메일 제목: `산방식당 파트너십 문의 - [날짜 시간]`

이메일 본문:
```

--- 문의 내용 ---

hong@example.com

010-1234-5678

문의유형:
메뉴솔루션 문의

---
이 이메일은 구글 폼에서 자동으로 전송되었습니다.
```

## 문제 해결

### 이메일이 오지 않는 경우

1. **스팸 폴더 확인**: 이메일이 스팸 폴더로 분류되었을 수 있습니다.

2. **트리거 확인**: Apps Script에서 트리거가 제대로 설정되었는지 확인합니다.

3. **권한 확인**: Apps Script에서 이메일 전송 권한이 승인되었는지 확인합니다.

4. **로그 확인**: Apps Script 실행 로그에서 오류 메시지를 확인합니다.

### Apps Script 실행 오류

- **"MailApp.sendEmail 권한 없음"**: 권한을 다시 승인해야 합니다.
- **"함수를 찾을 수 없음"**: 함수 이름이 정확한지 확인합니다 (`onFormSubmit`).

## 참고

- 구글 폼의 응답은 구글 시트에도 자동으로 저장됩니다.
- 방법 1과 방법 2를 동시에 사용할 수 있습니다 (중복 알림).
- 방법 2는 더 상세한 이메일 본문을 제공합니다.
